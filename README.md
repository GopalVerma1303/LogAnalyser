# Proxy Server for BHEL Hwr using Node.js
## Setup guide:
1. Clone the repo:
   ```bash
   git clone https://github.com/GopalVerma1303/BHELProxyServer.git
   ```
2. Open dir:
   ```bash
   cd BHELProxyServer
   ```
3. Run server.js
   ```bash
   node server.js
   ```
4. For MacOS:
   >Settings > Network > Proxies
   
   >Enable both **"Web Proxy"** and **"Secure Web Proxy"**
